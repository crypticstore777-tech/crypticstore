

## Create All-Over Print Apparel Products

### What
Create a line of all-over print apparel products on Shopify using the uploaded question mark pattern (black/white/yellow checkerboard with "?" symbols) — fits the Cryptic Store "MC Myster?ous" brand perfectly.

### Products to Create
Using the uploaded pattern image for all products:

1. **Myster?ous All-Over Print T-Shirt** — $34.99, sizes S-2XL
2. **Myster?ous All-Over Print Hoodie** — $59.99, sizes S-2XL
3. **Myster?ous All-Over Print Joggers** — $49.99, sizes S-2XL
4. **Myster?ous All-Over Print Shorts** — $39.99, sizes S-2XL
5. **Myster?ous All-Over Print Leggings** — $44.99, sizes S-2XL

All tagged with `alloverprint`, `Dare2Wear`, and `Cryptic Store` so they appear in relevant collection pages.

### How
1. Use `shopify--create_product` to create each product with:
   - The uploaded question mark pattern as the product image
   - Size options (S, M, L, XL, 2XL)
   - Price variants for each size
   - Proper tags and product type
   - Vendor set to "Cryptic Store"
2. Products will automatically appear in the store via the existing Shopify integration

### Notes
- These are print-on-demand products — you'll connect them to your fulfillment provider (Printful/Printify) via Shopify Admin
- The pattern image serves as the product listing photo; actual all-over print mockups can be swapped in later
